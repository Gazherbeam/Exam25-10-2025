Common HTML Tags Reference

Structure Tags

  • <html> - Root element of an HTML document.
  • <head> - Contains meta-information about the document.
  • <body> - Encloses the content of the document.
  • <header> - Defines the header section of a document or section.
  • <main> - Represents the main content of the document.
  • <footer> - Contains footer information for a document or section.
  • <nav> - Defines navigation links.
  • <section> - Represents a thematic grouping of content.
  • <img> - Embeds an image in the document.
  • <p> - Defines a paragraph of text.
  • <a> - Creates a hyperlink to another resource.
  • <div> - A generic container for grouping content.

Content Tags

  • <p> - Defines a paragraph of text.
  • <h1> to <h6> - Define headings, with <h1> being the highest level.
  • <img> - Embeds an image; use the <img src="url"> attribute to specify the image source.
  • <a> - Creates a hyperlink; use href attribute to specify the link target.
  • <ul>/<li> - Defines an unordered list; use <li> for list items.
  • <table> - Creates a table; use <tr> for rows and <td> for cells.
  • <div> - A generic container for grouping content.
  • <span> - An inline container for text or other elements.
  • <a href=""> - Hyperlink; specify the destination URL in the href attribute.
